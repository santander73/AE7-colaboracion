# AE7 - Colaboración efectiva de un proyecto

Este proyecto practica un flujo básico de trabajo colaborativo con Git y GitHub:
- Repositorio local con commits.
- Repositorio remoto en GitHub.
- Creación y uso de ramas.
- Pull Request y merge a la rama principal.

## Funcionalidades implementadas
- Página `index.html` con un encabezado que simula una barra de navegación (navbar) y secciones internas.

## Ramas utilizadas
- `main`: rama principal del proyecto.
- `feature/navbar`: rama para implementar el encabezado/navbar.
